Politique de confidentialité

ConcoursPolice · Dernière mise à jour : juin 2026

1. Responsable du traitement

MACREZ — messmacr@gmail.com
Application ConcoursPolice, disponible sur Google Play Store.

2. Données collectées

L'Application collecte uniquement les données suivantes, stockées localement sur votre appareil (aucun serveur externe) :

  • Prénom et âge (saisis lors de l'onboarding)
  • Scores et résultats de quiz
  • Progression par matière et par question
  • Objectif quotidien, streak (série de jours)
  • Points XP et grade obtenu

Ces données ne sont jamais transmises à des tiers, sauf dans les cas décrits à l'article 4.

3. Finalité

Les données collectées servent exclusivement à personnaliser votre expérience de révision : afficher votre progression, calculer votre streak et vos points XP, et adapter les recommandations de révision.

4. Services tiers

L'Application intègre les services suivants, qui peuvent collecter des données techniques :

  • RevenueCat — gestion des achats in-app Premium. Collecte des données de transaction (revenuecat.com/privacy).
  • Expo / EAS — plateforme de déploiement. Peut collecter des identifiants d'appareil anonymes pour les notifications push (expo.dev/privacy).
  • Google Play Store — distribution (policies.google.com/privacy).

5. Conservation

Les données sont conservées localement sur votre appareil tant que l'Application est installée. Elles sont supprimées automatiquement lors de la désinstallation ou via le bouton « Réinitialiser mes données » dans l'onglet Plus.

6. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Accès : vos données sont consultables dans l'app (onglet Plus)
  • Rectification : modifiables via les paramètres
  • Suppression : via le bouton « Réinitialiser mes données »

Pour toute demande : messmacr@gmail.com

7. Sécurité

Les données sont stockées via AsyncStorage (stockage natif chiffré par le système Android). Aucune donnée personnelle n'est transmise sur Internet par l'Application elle-même.

8. Mineurs

L'Application n'est pas destinée aux enfants de moins de 13 ans. Aucune collecte intentionnelle de données relatives à des mineurs n'est effectuée.

9. Modifications

Cette politique peut être mise à jour. La version en vigueur est toujours accessible dans l'Application (onglet Plus → Politique de confidentialité).
